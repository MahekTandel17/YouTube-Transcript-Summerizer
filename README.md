# YouTube Transcript Summarizer

A web application that fetches YouTube video transcripts and generates concise summaries using NLP.

## 🚀 Features

- Fetch transcripts from any YouTube video with captions
- Generate concise summaries using Python NLP
- Clean, responsive web interface
- Database storage for user history
- RESTful API architecture

| CSS3 | Styling |

## 📦 Installation

### Prerequisites

- Node.js (v14+)
- Python (v3.7+)
- MySQL/PostgreSQL
- Git

   ```bash
   git clone https://github.com/MahekTandel17/YouTube-Transcript-Summerizer.git
   cd YouTube-Transcript-Summerizer
